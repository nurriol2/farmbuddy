Fruit Saplings

At your farming level, each patch should be growing Papayas. Alternatively, you can grow Pineapples.

Sapling Payment

Payment will depend on which saplings you choose.

Papaya Sapling
10 Pineapples each
Pineapple Sapling
10 Watermelons each

Steps

  1. This run must begin at the Grand Exchange
  2. Run Northeast to the Spirit Tree
  3. Teleport to the Gnome Stronghold
  4. Run East to the fruit tree patch
  5. Teleport to Tree Gnome Village using the Spirit Tree
  6. Follow Elkoy and run West to the fruit tree patch
  7. Teleport to Camelot
  8. Run Southeast to the mainland fruit tree patch
  9. Run due West toward the dock
  10. Charter a boat to Brimhaven for 480 (gp)
